A simple IP address and network information utility that helps you inspect IP details, subnets, and related network metadata.
​

Features
Accepts user-supplied IP addresses as input and validates them.
​

Displays key details such as network address, broadcast address, and usable host range (based on the implementation inside ipaddress.zip).
​

Shows clean, user-friendly output similar to the screenshots in this repository.
​

Packaged as a small standalone project for quick experimentation and learning.
​

Getting started
Clone the repository.

Extract ipaddress.zip into a working directory.
​

Open the project in your preferred IDE or run it according to the language/framework used inside the zip (for example, Python script, Java project, etc.).
​

Run the main entry file and provide an IP address when prompted.
​
